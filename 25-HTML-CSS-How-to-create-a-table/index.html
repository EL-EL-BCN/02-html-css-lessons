27 - HTML-CSS What HTML Meta Tags are Required


Some meta tags can help with SEO - Search Engine Optimisation on our website. Below is a list of metatags that appear in our head tags.

01. title - Defines the name of the website.

02. Charset="UTF-8" - Defines what characters can be used inside the website.

03. viewport - used when we have responsive websites that work on mobile, pad and desktop devices.

04. Description - describes the website. We cannot have more than 160 characters inside this meta tag.

06. author - who built the website etc.
