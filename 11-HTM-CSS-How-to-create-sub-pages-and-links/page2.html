In this lesson we are hoing to learn how to add more pages to our website.

To add a link we use an anchor tag denoted by an "a". Within the anchor tag we add a hyper reference, href, that has the name of the page we want to link to. If the page is in the root folder we need to also include the path to that page. E.g. ../index.html
