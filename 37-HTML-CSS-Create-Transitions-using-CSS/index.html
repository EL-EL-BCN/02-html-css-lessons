01. Transitions can be used to gradually change background colours, borders etc rather than abruptly.Below is the starting config.



02a. We can use a transition-property in CSS. we can set any property but in this case we set it to all.

02b. Transition-duration is the amount of time that it takes to transition from one image to the other, in this case two seconds.

02c. We can also set the transition-timing-function such as in steps or ease in and out.

02d. We can also add a transition-delay. In this case the transition will not start until the mouse has been hovering over the element for 4 seconds.



03. There could be compatability issues because some older versions of Chrome, Mozilla firefox, Opera etc may not support the trasitions CSS feature. So we have to use webkit for chrome, moz for firefox and o for opera.
